AgentGUI

Multi-agent GUI for Claude Code, Gemini CLI, OpenCode & more. Real-time streaming, dark mode, voice, and SQLite persistence.

View on GitHub npm Package

Get Started in Seconds

No installation required. Run with npx and start coding with AI agents instantly.

npx agentgui

Server starts at http://localhost:3000/gm/

Why AgentGUI?

Stop juggling multiple terminal windows. Work with all your AI agents in one unified interface.

📊 Compare Side-by-Side

Test the same prompt across Claude Code, Gemini CLI, OpenCode, and others to find the best approach for your task.

💾 Never Lose Context

Every conversation, file change, and terminal output is automatically saved. Resume interrupted work exactly where you left off.

👁️ See Everything

Watch streaming responses, file changes, and tool calls in real-time instead of parsing raw JSON in your terminal.

Features

Everything you need to work with multiple AI coding agents in one place.

🤖

Multi-Agent Support

Auto-discovers 14 AI coding agents including Claude Code, Gemini CLI, OpenCode, Kilo, Goose, Cline, Codex, Kiro, and more.

⚡

Real-Time Streaming

WebSocket-based streaming for instant responses. Watch your agent work in real-time with live execution visualization.

💾

Persistent Storage

SQLite database stores all conversations, sessions, and execution history. Resume where you left off.

📁

File Management

Integrated file browser with drag-and-drop uploads. Direct file editing and context-aware operations.

🎤

Speech Capabilities

Built-in speech-to-text and text-to-speech via Hugging Face Whisper. No API keys required.

🔧

Developer Friendly

Hot reload, REST API, WebSocket endpoints, and extensible plugin system for custom agents.

REST API

Complete REST API for conversations, messages, agents, tools, and speech operations.

WebSocket Sync

Real-time bidirectional communication with event-based streaming and subscription model.

SQLite WAL Mode

Write-Ahead Logging for concurrent reads and writes without blocking.

Installation Options

Quick Start (Recommended)

npx agentgui

Manual Installation

git clone https://github.com/AnEntrypoint/agentgui.git
cd agentgui
npm install
npm run dev

System Requirements

  • ✓ Node.js 18+ (LTS recommended)
  • ✓ npm or bun package manager
  • ✓ At least one AI coding agent (Claude Code, Gemini CLI, etc.)
