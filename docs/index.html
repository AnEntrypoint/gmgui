AgentGUI

Multi-agent GUI for Claude Code, Gemini CLI, OpenCode & more.
Real-time streaming, dark mode, voice, and SQLite persistence.

View on GitHub npm Package

Get Started in Seconds

No installation required. Run with npx and start coding with AI agents instantly.

$ npx agentgui

Server starts at http://localhost:3000/gm/

Why AgentGUI?

Stop juggling multiple terminal windows. One interface for all your AI agents.

📊 Compare Side-by-Side

Test the same prompt across Claude Code, Gemini CLI, OpenCode, and others to find the best approach.

💾 Never Lose Context

Every conversation, file change, and terminal output is automatically saved. Resume exactly where you left off.

👁️ See Everything

Watch streaming responses, file changes, and tool calls in real-time instead of parsing raw JSON.

Features

Everything you need to work with multiple AI coding agents in one place.

🤖

Multi-Agent Support

Auto-discovers 14 AI coding agents: Claude Code, Gemini CLI, OpenCode, Kilo, Goose, Cline, Codex, Kiro, and more.

⚡

Real-Time Streaming

WebSocket-based streaming for instant responses. Watch your agent work live with execution visualization.

💾

Persistent Storage

SQLite WAL mode stores all conversations, sessions, and execution history. Resume where you left off.

📁

File Management

Integrated file browser with drag-and-drop uploads. Direct file editing and context-aware operations.

🎤

Voice I/O

Built-in speech-to-text and text-to-speech via Hugging Face Whisper. No API keys required.

🔧

Developer Friendly

Hot reload, REST API, WebSocket endpoints, and extensible plugin system for custom agents.

Installation Options

Quick Start (Recommended)

$ npx agentgui

Manual Installation

$ git clone https://github.com/AnEntrypoint/agentgui.git
$ cd agentgui
$ npm install
$ npm run dev

System Requirements

✓ Node.js 18+ (LTS recommended)
✓ npm or bun package manager
✓ At least one AI coding agent (Claude Code, Gemini CLI, etc.)
